# Face Recognizer using OpenCV

## Overview
This project detects and recognizes faces using OpenCV and Haar Cascade Classifiers. It captures video from a webcam, detects faces in real-time, and displays the detected faces with labels.

## Features
- Real-time face detection
- Webcam video capture
- Face recognition using trained data
- Draws rectangles around detected faces
- Displays person's name on recognized faces

## Requirements
- Python 3.x
- OpenCV
- NumPy
